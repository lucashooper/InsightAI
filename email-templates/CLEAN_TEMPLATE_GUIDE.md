## 📏 **Specifications:**

### **Container:**
- Max width: 500px (compact like Whop)
- Background: #ffffff (white)
- Border radius: 16px
- Shadow: 0 4px 12px rgba(0, 0, 0, 0.08)

### **Logo:**
- Size: 64px circle
- Gradient: #8b5cf6 → #ec4899
- Icon: 👁️ (32px)

### **Typography:**
- Brand name: 24px, 600 weight
- Greeting: 24px, 600 weight
- Message: 15px, 1.6 line-height
- Color: #1a1a1a (heading), #666666 (body)

### **Button:**
- Padding: 14px × 32px
- Gradient: #8b5cf6 → #a78bfa
- Border radius: 10px
- Shadow: 0 4px 12px rgba(139, 92, 246, 0.3)

### **Colors:**
- Background: #f5f5f5 (light gray)
- Card: #ffffff (white)
- Text: #1a1a1a (dark), #666666 (gray)
- Button: Purple gradient
- Links: #8b5cf6

## 🚀 **How to Use:**

### **Step 1: Update Supabase**
1. Go to [Supabase Dashboard](https://supabase.com/dashboard)
2. Authentication → Email Templates
3. Click "Confirm signup"
4. Copy contents of `confirm-signup-clean.html`
5. Paste and **Save**

### **Step 2: Test**
1. Sign up with a test email
2. Check inbox
3. ✅ See the clean, minimal email!
